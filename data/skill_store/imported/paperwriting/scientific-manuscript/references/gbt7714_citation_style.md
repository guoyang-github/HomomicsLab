# GB/T 7714-2015 参考文献著录规则

## 概述

GB/T 7714-2015 是中国国家标准《信息与文献 参考文献著录规则》，由中国国家标准化管理委员会发布，是中国学术论文、学位论文、科技报告等文献引用格式的统一规范。几乎所有国内期刊和高校学位论文都要求使用此标准。

**与西方标准的区别**：
- GB/T 7714 采用"[文献类型标识]"系统（如 `[J]` 期刊，`[M]` 专著）
- 支持中英文文献混排，英文文献保持英文格式
- 作者姓名采用"姓前名后"或"姓全拼，名缩写"两种形式
- 日期格式统一为 `YYYY-MM-DD` 或 `YYYY`

---

## 两种著录体系

### 1. 顺序编码制（Numeric）

**特点**：按引用先后顺序用阿拉伯数字编号，置于方括号中。

**正文中的引用格式**：
```
张三等[1]首次发现了这一现象。

后续研究[2-4]证实了该结论。

研究表明[1,3,5-7]，该机制具有普遍性。
```

**上标格式**（部分期刊要求）：
```
张三等¹首次发现了这一现象。

后续研究²⁻⁴证实了该结论。
```

**参考文献列表格式**：
按正文中引用顺序排列，编号左顶格，后续行缩进2字符。

```
[1] 作者. 题名[文献类型标识]. 其他信息.
[2] 作者. 题名[文献类型标识]. 其他信息.
```

### 2. 著者-出版年制（Author-Year）

**特点**：正文中标注作者姓氏和出版年，参考文献按作者字母顺序排列。

**正文中的引用格式**：
```
张三等 (2020) 首次发现了这一现象。

这一机制已被广泛证实 (张三等, 2021; Li et al, 2022)。

Wang et al (2019) 和李四 (2020) 分别从不同角度进行了研究。
```

**参考文献列表格式**：
按作者姓氏汉语拼音/英文字母顺序排列。

```
张三, 李四. 2020. 题名[文献类型标识]. 其他信息.
Li W, Wang H. 2022. Title[文献类型标识]. Other information.
```

## 各类文献著录格式详解

### 1. 期刊论文 `[J]`

**中文期刊**：
```
[序号] 主要责任者. 题名[文献类型标识]. 刊名, 年, 卷(期): 起止页码.

[1] 张三, 李四, 王五. 新型催化材料的研究进展[J]. 化学学报, 2023, 81(5): 456-463.
[2] 赵六, 钱七. 基于深度学习的图像识别方法[J]. 计算机学报, 2024, 47(2): 301-315.
```

**英文期刊**：
```
[序号] 主要责任者. 题名[文献类型标识]. 刊名, 年, 卷(期): 起止页码.

[3] SMITH J A, JOHNSON B C, WILLIAMS K R. Novel catalytic materials for energy conversion[J]. Nature Chemistry, 2023, 15(3): 234-241.
[4] LI W, ZHANG H, LIU L. Deep learning approaches for medical image analysis[J]. IEEE Transactions on Medical Imaging, 2024, 43(1): 89-102.
```

**注意**：
- 作者超过3人时，中文文献用"等"，英文文献用"et al"
- 卷号加粗（部分期刊要求）
- DOI 可加在页码后（可选）：`456-463. DOI: 10.xxxx/xxxxx`

---

### 2. 专著 `[M]`

**中文专著**：
```
[序号] 主要责任者. 题名[文献类型标识]. 版本项. 出版地: 出版者, 出版年: 引文页码.

[5] 黄昆, 韩汝琦. 固体物理学[M]. 2版. 北京: 高等教育出版社, 2020: 156-160.
[6] 张三. 人工智能导论[M]. 北京: 清华大学出版社, 2023.
```

**翻译著作**：
```
[7] 原作者. 书名[M]. 译者, 译. 出版地: 出版者, 出版年.

[8] KNUTH D E. 计算机程序设计艺术[M]. 苏运霖, 译. 北京: 国防工业出版社, 2019.
```

**英文专著**：
```
[9] SMITH J A, JOHNSON B C. Advanced Materials Science[M]. 3rd ed. New York: Academic Press, 2022: 245-267.
```

---

### 3. 学位论文 `[D]`

**中文学位论文**：
```
[序号] 主要责任者. 题名[文献类型标识]. 保存地: 保存单位, 年份.

[10] 王十二. 基于石墨烯的储能材料研究[D]. 北京: 清华大学, 2023.
[11] 李四. 深度学习在医学影像中的应用研究[D]. 上海: 复旦大学, 2024.
```

**英文学位论文**：
```
[12] WILLIAMS K R. Novel Approaches to Catalytic Materials[D]. Cambridge: Massachusetts Institute of Technology, 2023.
```

---

### 4. 会议论文 `[C]`

**中文会议论文**：
```
[序号] 主要责任者. 题名[文献类型标识]//编者. 会议论文集名. 出版地: 出版者, 出版年: 起止页码.

[13] 张三, 李四. 一种新型图像分类算法[C]//中国计算机学会. 2023中国计算机大会论文集. 北京: 电子工业出版社, 2023: 123-130.
```

**英文会议论文**：
```
[14] SMITH J A, JOHNSON B C. A novel approach to image classification[C]//Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition. Los Alamitos: IEEE Computer Society, 2023: 567-574.
```

---

### 5. 专利 `[P]`

```
[序号] 专利所有者. 专利题名: 专利号[文献类型标识]. 公告日期或公开日期.

[15] 张三, 李四. 一种高效太阳能电池制备方法: CN202310123456.7[P]. 2023-05-15.
[16] SMITH J A. Method for preparing high-efficiency solar cells: US10,123,456 B2[P]. 2023-06-20.
```

---

### 6. 标准 `[S]`

```
[序号] 标准编号, 标准名称[文献类型标识].

[17] GB/T 7714-2015, 信息与文献 参考文献著录规则[S].
[18] GB 3100-1993, 国际单位制及其应用[S].
```

---

### 7. 电子文献/在线文献 `[EB/OL]` 或 `[J/OL]`

```
[序号] 主要责任者. 题名[文献类型标识/文献载体标识]. (更新或修改日期)[引用日期]. 获取和访问路径.

[19] 张三. 中国科技发展报告[EB/OL]. (2023-12-01)[2024-03-15]. http://www.example.com/report.
[20] SMITH J A, JOHNSON B C. Supplementary data for novel materials[J/OL]. Nature Materials, 2023[2024-01-10]. https://doi.org/10.1038/s41563-023-01678-x.
```

---

### 8. 报告 `[R]`

```
[序号] 主要责任者. 题名[文献类型标识]. 出版地: 出版者, 出版年.

[21] 中国科学院. 2023科学发展报告[R]. 北京: 科学出版社, 2023.
[22] WORLD HEALTH ORGANIZATION. World Health Report 2023[R]. Geneva: WHO, 2023.
```

## 中英文文献混排规则

### 排序规则（顺序编码制）
按引用出现顺序排列，不区分中英文。

### 排序规则（著者-出版年制）
按作者姓氏排序：
1. 中文作者按姓氏拼音排序（如 李/Li, 王/Wang, 张/Zhang）
2. 英文作者按字母顺序排序
3. 同一作者多篇文献按出版年排序

### 作者姓名格式

**中文文献**：
- 3人以内：全部列出，`张三, 李四, 王五`
- 超过3人：`张三, 等`（顺序编码制）或 `张三等`（著者-出版年制）

**英文文献**：
- 3人以内：全部列出，`SMITH J A, JOHNSON B C, WILLIAMS K R`
- 超过3人：`SMITH J A, et al`

**注意**：
- 姓氏全大写（部分期刊要求）
- 名用首字母缩写
- 作者之间用逗号分隔

## 标点符号规范

| 位置 | 符号 | 说明 |
|------|------|------|
| 作者之间 | `, ` | 逗号+空格 |
| 作者与题名之间 | `. ` | 句点+空格 |
| 题名与文献类型标识之间 | `[` `]` | 方括号 |
| 刊名与年份之间 | `, ` | 逗号+空格 |
| 年份与卷号之间 | `, ` | 逗号+空格 |
| 卷号与期号之间 | `(` `)` | 圆括号 |
| 期号与页码之间 | `: ` | 冒号+空格 |
| 页码范围 | `-` | 短横线 |
| 多条文献 | `; ` | 分号+空格 |

## 常见错误与注意事项

### 1. 文献类型标识遗漏
- ❌ `张三. 论文标题. 期刊名, 2023.`
- ✅ `张三. 论文标题[J]. 期刊名, 2023.`

### 2. 作者姓名格式错误
- ❌ `Smith J, Johnson B`
- ✅ `SMITH J A, JOHNSON B C`（姓氏全大写）

### 3. 中英文标点混用
- ❌ `张三，李四。论文标题[J]。期刊名，2023，45（2）：123-130。`
- ✅ `张三, 李四. 论文标题[J]. 期刊名, 2023, 45(2): 123-130.`

### 4. 页码格式不统一
- ❌ `123—130` 或 `123~130`
- ✅ `123-130`（短横线）

### 5. 缺少出版地（专著）
- ❌ `张三. 书名[M]. 出版社, 2023.`
- ✅ `张三. 书名[M]. 北京: 出版社, 2023.`

### 6. 电子文献缺少引用日期
- ❌ `张三. 报告[EB/OL]. http://...`
- ✅ `张三. 报告[EB/OL]. (2023-01-01)[2024-03-15]. http://...`
