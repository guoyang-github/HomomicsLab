# 中国学位论文格式规范

## 概述

学位论文是申请硕士或博士学位必须提交的核心学术成果，其格式规范由各高校根据国家标准制定。本指南汇总了国内高校学位论文的通用格式要求，涵盖从封面到致谢的完整结构。

**主要依据标准**：
- GB/T 7713.1-2006《学位论文编写规则》
- GB/T 7714-2015《信息与文献 参考文献著录规则》
- 各高校学位论文格式要求（通常在研究生院网站公布）

## 各部分格式要求

### 1. 封面

**内容**（各校统一格式）：
- 学校名称（校徽）
- 学位论文中英文标题
- 作者姓名
- 指导教师姓名、职称
- 学科专业
- 研究方向
- 培养单位
- 完成日期

**格式要求**：
- 学校提供统一模板，必须严格按照模板填写
- 论文标题一般不超过 25 个汉字
- 标题应准确反映论文核心内容

---

### 2. 原创性声明和版权授权书

**内容**：
- 原创性声明：声明论文为本人独立完成，未抄袭他人成果
- 版权授权书：授权学校使用论文进行存档、检索和发布

**要求**：
- 必须手写签名
- 日期需填写完整
- 不得修改模板内容

---

### 3. 中文摘要

**要求**：
- **博士学位论文**：1000-1500 字
- **硕士学位论文**：500-800 字
- **关键词**：3-8 个

**内容要素**：
1. 研究背景和目的（1-2句）
2. 研究方法（2-3句）
3. 主要研究结果（重点，占60%篇幅）
4. 研究结论和创新点（2-3句）

**写作建议**：
- 用第三人称撰写
- 不使用图表和公式
- 不引用参考文献
- 语言简洁准确

---

### 4. 英文摘要（Abstract）

**要求**：
- **博士**：300-500 words
- **硕士**：200-300 words
- **Keywords**：3-8 个

**注意**：
- 应与中文摘要内容对应
- 使用被动语态和过去时态描述研究工作
- 不使用缩写（首次出现需全称）

---

### 5. 目录

**格式要求**：
- 自动生成（Word/LaTeX 目录功能）
- 显示到三级标题（1.1.1）
- 页码右对齐
- 目录中不显示图片和表格编号

**示例**：
```
目录

摘要 .................................................... I
Abstract ............................................... III
第1章 绪论 ............................................. 1
  1.1 研究背景与意义 ................................... 1
    1.1.1 研究背景 ................................... 1
    1.1.2 研究意义 ................................... 3
  1.2 国内外研究现状 ................................... 5
  1.3 研究内容与论文结构 ............................... 8
第2章 理论基础 ......................................... 10
...
参考文献 ............................................... 120
致谢 ................................................... 130
```

---

### 6. 正文格式

#### 6.1 章节编号

采用阿拉伯数字分级编号：
```
第1章  ××××
  1.1  ××××
    1.1.1  ××××
      1. ××××（可选，部分学校允许四级标题）
```

**注意**：
- "第X章"后空一格接标题
- 一级标题（章）另起一页
- 一般不超过四级标题

#### 6.2 字体字号（通用规范）

| 层级 | 字体 | 字号 | 格式 |
|------|------|------|------|
| 封面标题 | 黑体 | 小一或二号 | 居中 |
| 章标题 | 黑体 | 三号（16pt）| 居中 |
| 节标题 | 黑体 | 四号（14pt）| 左对齐 |
| 小节标题 | 黑体 | 小四（12pt）| 左对齐 |
| 正文 | 宋体 | 小四（12pt）| 两端对齐 |
| 图表标题 | 黑体/宋体 | 五号（10.5pt）| 居中/左对齐 |

**注意**：
- 英文和数字使用 Times New Roman
- 行距：1.5倍或固定值 20-22 磅
- 页边距：上 2.5-2.8cm，下 2.2-2.5cm，左 2.8-3.0cm，右 2.2-2.5cm
- 页眉：学校名称或论文题目，五号字
- 页脚：页码，居中或右对齐

#### 6.3 段落格式

- 首行缩进 2 字符（约 0.74cm）
- 段前段后间距：0 或 6pt
- 正文段落不宜过长，建议 3-5 行为一段

---

### 7. 图表规范

#### 7.1 图（Figure）

**编号格式**：
```
图1-1  图的标题
图2-3  图的标题
```
- 章号-图序号（如第1章第1个图为"图1-1"）
- 图标题置于图的下方
- 图标题居中，五号宋体

**格式要求**：
- 图中文字使用宋体或 Times New Roman
- 图的大小适中，清晰可读
- 彩色图需确保黑白打印时仍可辨识
- 分辨率：≥300 DPI

#### 7.2 表（Table）

**编号格式**：
```
表2-1  表的标题
表3-4  表的标题
```
- 章号-表序号
- 表标题置于表的上方
- 表标题居中，五号宋体

**格式要求**：
- 三线表为主（顶线、栏目线、底线）
- 表内文字使用五号或小五号
- 单位统一标注在栏目名称中

---

### 8. 公式规范

**编号格式**：
```
E = mc²    (1-1)

∇ · E = ρ/ε₀    (2-3)
```
- 章号-公式序号
- 公式居中，编号右对齐
- 使用 MathType 或 LaTeX 编辑

---

### 9. 参考文献

**格式**：GB/T 7714-2015

**数量要求**：
- 博士学位论文：通常 100-200 篇
- 硕士学位论文：通常 50-100 篇

**要求**：
- 中英文文献均可
- 近 5 年文献应占 50% 以上
- 必须有适量英文文献
- 按引用顺序排列（顺序编码制）

详见 `references/gbt7714_citation_style.md`

---

### 10. 页码设置

| 部分 | 页码格式 | 起始页码 |
|------|---------|---------|
| 封面至目录 | 不编页码或单独编排（罗马数字） | — |
| 正文至结尾 | 阿拉伯数字 | 1 |

## 各章节写作要点

### 第1章 绪论

**内容框架**：
1. **研究背景**（2-3页）
   - 研究领域概述
   - 问题的提出
   
2. **研究意义**（1-2页）
   - 理论意义
   - 实际应用价值
   
3. **国内外研究现状**（博士 10-20 页，硕士 5-10 页）
   - 系统文献综述
   - 指出研究空白
   
4. **研究内容与创新点**（1-2页）
   - 论文主要工作
   - 创新点总结
   
5. **论文组织结构**（0.5-1页）

### 研究内容章节（第2章至第N章）

每章结构建议：
```
2.1 引言（简述本章研究内容）
2.2 [核心理论/方法]
2.3 [实验/实现]
2.4 [结果与分析]
2.5 本章小结
```

### 结论与展望

**结论**（总结研究成果，不重复摘要）：
- 概括主要研究发现
- 强调创新贡献
- 指出研究的局限性

**展望**（未来工作方向）：
- 可改进之处
- 后续研究方向

## LaTeX 模板建议

推荐使用 `ctex` 宏包编写中文学位论文：

```latex
\documentclass[UTF8,a4paper,12pt]{ctexart}
% 或使用学校提供的模板

\usepackage{geometry}
\geometry{top=2.5cm,bottom=2.5cm,left=3cm,right=2.5cm}

\usepackage{times}  % Times New Roman for English
\setCJKmainfont{SimSun}  % 宋体
\setCJKsansfont{SimHei}  % 黑体

\ctexset{
  chapter = {
    format = \zihao{3}\heiti\centering,
    beforeskip = 24pt,
    afterskip = 18pt
  },
  section = {
    format = \zihao{4}\heiti,
    beforeskip = 12pt,
    afterskip = 6pt
  }
}

\begin{document}
  % 论文内容
\end{document}
```

## 常见格式错误

| 错误类型 | 错误示例 | 正确做法 |
|---------|---------|---------|
| 字体混用 | 正文用楷体 | 正文统一用宋体 |
| 字号不统一 | 同层级标题字号不同 | 严格按照层级设置字号 |
| 行距不一致 | 部分段落单倍，部分1.5倍 | 全文统一行距 |
| 页边距不对 | 左右边距相同 | 左边距（装订侧）略大 |
| 图表编号混乱 | 图1.1、Figure 1.1 混用 | 全文统一为"图1-1" |
| 参考文献格式不统一 | 部分 GB/T 7714，部分 APA | 全文统一用 GB/T 7714 |
| 页码遗漏 | 封面后开始编页码 | 封面不编页码或单独编排 |

## 检查清单

提交前逐项检查：

- [ ] 封面信息完整、准确
- [ ] 原创性声明和授权书已签名
- [ ] 中英文摘要对应，关键词一致
- [ ] 目录页码正确
- [ ] 正文格式统一（字体、字号、行距、页边距）
- [ ] 图表编号连续，标题完整
- [ ] 公式编号正确
- [ ] 参考文献格式统一，数量达标
- [ ] 页码设置正确
- [ ] 无错别字和语法错误
- [ ] 已按学校模板调整格式

---

**注意**：不同高校的具体格式要求可能有差异（如字体、字号、页边距等），务必以所在学校的《学位论文撰写规范》为准。本指南提供的是通用规范，具体细节请查阅学校研究生院网站。
