# Project Timeline and Gantt Chart Template

## Overview

Use this template to create clear, compelling timelines for grant proposals. Adapt the structure to your specific project aims and agency requirements.

### Milestone Summary

| Milestone | Target Date | Success Criteria | Go/No-Go Criteria |
|-----------|-------------|------------------|-------------------|
| Method validated | End Y1 | Reproducibility >95% | If <80%, pivot to alternative approach |
| Data collection complete | Mid Y2 | N=100 per group | If recruitment <50%, expand to additional sites |
| Integration complete | Mid Y3 | Statistical model converges | If models fail, use simplified approach |
| Manuscripts submitted | End Y3 | 2 papers in submission | If results negative, publish methods paper |

## Tips for Creating Effective Timelines

1. **Show Parallel Activities**: Multiple aims should proceed simultaneously where possible
2. **Include Dependencies**: Make clear which tasks depend on others
3. **Be Realistic**: Build in 10-20% buffer for unexpected delays
4. **Highlight Milestones**: Clear decision points show project management
5. **Match Budget**: Timeline personnel should align with budgeted effort
6. **Account for Seasons**: Field work, academic calendars, conference schedules

## Adaptations by Agency

### NSF
- Integrate broader impacts activities throughout timeline
- Show student training milestones
- Include data sharing and dissemination activities

### NIH
- Align with typical 5-year R01 structure (extend table)
- Include study section review cycles for potential resubmission
- Show preliminary data generation in early years

### DARPA
- Emphasize phase-based structure
- Include quarterly technical milestones with metrics
- Show demonstration and transition events

### DOE
- Align with fiscal year reporting
- Include facility access scheduling
- Show industry partnership activities
