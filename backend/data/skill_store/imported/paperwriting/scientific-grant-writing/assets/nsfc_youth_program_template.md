## 青年项目特别注意事项

### 写作策略差异

1. **更强调创新潜力**：评委更关注申请人的创新思维和独立研究潜力，而非已有的工作积累
2. **基础可以相对薄弱**：可以坦诚说明前期基础有限，但要有清晰的研究思路
3. **导师角色**：可在工作条件中提及导师/团队的支持，但需强调项目的独立性
4. **目标要务实**：3年30万，不宜设置过于宏大的目标
5. **体现成长性**：展示从博士工作到独立研究的过渡和突破

### 评审关注点

- 申请人是否具备独立开展科研工作的潜力
- 研究方向是否具有创新性
- 研究方案是否合理可行
- 依托单位是否提供足够支持
