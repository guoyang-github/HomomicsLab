# 国内权威期刊投稿指南

## 概述

中国拥有众多高水平学术期刊，涵盖自然科学、工程技术、医学、社会科学等各个领域。本指南汇总了国内最具代表性的权威期刊的投稿要求，帮助作者快速了解各期刊的格式规范和投稿流程。

**国内期刊分级**（参考）：
- **T类**：国际顶尖期刊（如《中国科学》系列部分英文刊达到国际一流水平）
- **A类**：国内顶级期刊（《中国科学》《科学通报》及各学科权威期刊）
- **B类**：国内高水平期刊（各学科核心期刊）
- **C类**：普通核心期刊

## 综合性期刊

### 《中国科学》系列（Science China）

**简介**：中国科学院主办，中国最具国际影响力的综合性学术期刊，分中英文两个版本，涵盖数学、物理学、化学、生命科学、地球科学、信息科学、技术科学等多个学科。

| 项目 | 要求 |
|------|------|
| **投稿网址** | https://www.sciengine.com/ 或 https://www.scichina.com/ |
| **语言** | 中文刊（中文投稿，英文摘要）/ 英文刊（Science China 系列，全英文） |
| **稿件类型** | 研究论文、综述、快报、进展 |
| **摘要要求** | 中文摘要 300-500字，英文摘要 200-300 words，均需中英文关键词（3-8个） |
| **参考文献** | GB/T 7714 格式，建议 20-40 篇 |
| **图表要求** | 彩色图免费，要求高清（≥300 DPI） |
| **审稿周期** | 约 1-3 个月 |
| **版面费** | 有，具体金额咨询编辑部 |
| **录用率** | 约 10-20% |

**特色要求**：
- 强调原创性和学术价值
- 英文刊（Science China 系列）要求较高，对标国际一流期刊
- 鼓励跨学科研究

## 国内期刊 vs 国际期刊投稿对比

| 维度 | 国内期刊 | 国际期刊 |
|------|---------|---------|
| **语言** | 中文为主 + 英文摘要 | 英文 |
| **引用格式** | GB/T 7714 | APA / AMA / Vancouver / Nature 等 |
| **审稿周期** | 通常 1-3 个月 | 通常 2-6 个月 |
| **版面费** | 多数收取 | 开源期刊收取 APC，传统期刊多免版面费 |
| **影响因子** | 部分有，但整体偏低 | 普遍有，部分较高 |
| **评审标准** | 强调应用价值和国内需求 | 强调创新性和国际影响力 |
| **学术声誉** | 在国内认可度高 | 国际认可度高 |

## 投稿策略建议

### 1. 初次投稿建议
- **如果创新性高、具有国际影响力**：优先投国际期刊
- **如果应用性强、符合国内需求**：国内权威期刊是很好选择
- **如果课题组要求/毕业要求**：根据要求选择相应级别期刊

### 2. 国内期刊优势
- 审稿周期通常更短
- 对国内研究背景更了解
- 应用价值更容易被认可
- 版面费通常较低

### 3. 提升录用率的建议
- 严格按照期刊格式要求撰写
- 强调研究对学科发展或国家需求的贡献
- 参考文献中适当引用该期刊的论文
- 请国内同行预审

## 国内期刊分级参考

### 北大核心（北京大学图书馆《中文核心期刊要目总览》）
- 每 3-4 年评选一次
- 涵盖 70 多个学科门类
- 国内最广泛认可的核心期刊评价体系

### 南大核心（CSSCI / 中文社会科学引文索引）
- 南京大学中国社会科学研究评价中心开发
- 主要针对人文社会科学期刊
- 分来源期刊和扩展版来源期刊

### CSCD（中国科学引文数据库）
- 中国科学院文献情报中心开发
- 主要针对自然科学领域
- 分核心库和扩展库

### 中国科技核心（中国科技论文统计源期刊）
- 中国科学技术信息研究所评选
- 涵盖自然科学和工程技术领域
